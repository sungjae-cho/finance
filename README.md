# My finance project

- Financial management for stock portfolios
  - Multiple portfolios.
  - Adaptiable to stock splits.
  - Dividends are included as captial returns.
  - Stocks from multiple stock markets with multiple currencies.
  - Ability to compute and visualize the performance of portfolios within a specified period
    - Total return: The capital return until the end of a specified period.
    - Period return: The capital return wihin a specified period. Though this, you can measure the performance of annual returns.
